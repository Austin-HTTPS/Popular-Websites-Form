About

This website is a simple program that collects popular websites. I'm creating an internet based game, and I want to implement some of these sites. The problem is, I can only come up with so many of them before I run out of ideas. Because of this, I created this program that verifies and collects fun, popular websites. I will look through this every now and then, and have created a check to make sure that the websites exist. Please keep these sites SFW.
